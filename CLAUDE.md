# BankReserves

## Overview
FDIC bank data calculator and dashboard. Displays bank financial data, reserve ratios, and risk metrics.

## Classification
- **Tier:** 2 — Interactive
- **Company:** GetDigital

## Tech Stack
- **Framework:** Next.js 15.5, React 18
- **Language:** TypeScript 5.4
- **Styling:** Tailwind CSS
- **Package Manager:** npm

## Key Commands
- `npm run dev` — Start development server
- `npm run build` — Production build
